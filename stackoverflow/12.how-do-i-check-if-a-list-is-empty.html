12. 如何检查列表是否为空

题目

例如，如果通过以下内容:

a = []

我如何检查a是否为空?

链接

回答一

if not a:
    print("List is empty")

使用空列表的隐式booleanness是相当python的（Pythonic）。

回答二

python风格的方法是来自PEP 8风格指南(在这里，Yes的意思是“推荐”，而NO是“不推荐”): 对于序列，(字符串、列表、元组)，使用“空序列为假”的这一事实

Yes: if not seq:
     if seq:

No:  if len(seq):
     if not len(seq):
